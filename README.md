# XHS Obsidian Workflow Starter

Free starter templates for turning an Obsidian note into a Xiaohongshu post package.

This repository is for creators who already keep drafts, course notes, interview notes, or reusable ideas in Obsidian, but still rebuild the Xiaohongshu publishing workflow by hand every time.

## What This Starter Gives You

- A reusable Obsidian note template for one Xiaohongshu post.
- A collection inbox template for turning saved Xiaohongshu notes into source cards.
- A release checklist for title, body, carousel pages, image prompts, tags, and publishing order.
- A sample output structure for keeping every post package easy to review later.
- A simple boundary: this is a workflow scaffold, not a traffic guarantee or account operation service.

## Quick Start

1. If your Xiaohongshu saved notes are messy, copy `templates/xhs-collection-inbox.md` into your Obsidian vault first.
2. Move 5-20 saved notes into source cards and tag them as `usable`, `reference`, or `discard`.
3. Copy `templates/xhs-post-brief.md` when one source card is ready to become a post.
4. Fill in the source material, reader, pain point, promise, and proof.
5. Use `checklists/release-checklist.md` before generating images or publishing.
6. Archive the final post package with the structure in `examples/sample-output-structure.md`.

## Recommended Workflow

```text
Saved Xiaohongshu note
  -> collection inbox
  -> source card
  -> content brief
  -> title candidates
  -> body draft
  -> carousel page plan
  -> image prompts
  -> publishing draft
  -> archive package
```

Each post should stay traceable. When a post performs well or badly, you should be able to reopen the folder and see the original source, the title decision, the image prompts, and the final draft.

## Good Fit

- You already use Obsidian for notes, drafts, course material, or content ideas.
- You publish or plan to publish Xiaohongshu carousel posts repeatedly.
- You want a repeatable production path from source material to final publishing package.
- You can bring your own AI model account, image generation account, and Xiaohongshu account.

## Not a Good Fit

- You want someone to operate your Xiaohongshu account for you.
- You expect guaranteed views, followers, or sales.
- You do not want to configure any tool or read a workflow document.
- Your source material is still too thin to become useful posts.
