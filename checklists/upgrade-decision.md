# Upgrade Decision Checklist

Use this checklist before buying the full pack. The goal is to avoid selling the wrong thing to the wrong user.

## Use the Free Starter If

- You only need a folder structure and a post brief template.
- You are comfortable writing title candidates, body drafts, and image prompts manually.
- You already have your own Obsidian workflow and only want a small publishing checklist.
- You do not need a packaged plugin or step-by-step setup guide.

## Consider the Full Pack If

- You already keep source material in Obsidian and publish Xiaohongshu posts repeatedly.
- You want one repeatable path from source note to title, body, carousel plan, image prompts, final post, and archive package.
- You want the packaged Obsidian desktop plugin and a first-run guide in one bundle.
- You want sample post packages to compare against your own output.
- You can bring your own AI model account, image generation account, API key, and Xiaohongshu account.

## Do Not Buy If

- You expect guaranteed views, followers, sales, or income.
- You want someone to operate your Xiaohongshu account.
- You want remote installation, account login help, or API key setup done for you.
- You do not currently have notes, drafts, course material, transcripts, or reusable source material.
- You are not willing to read a setup guide or run one manual test package.

## Before Buying

1. Open `examples/complete-mini-run.md`.
2. Try to turn one real Obsidian note into the sample package structure.
3. Use `checklists/release-checklist.md` to review the result.
4. If the manual flow already solves your problem, keep using the free starter.
5. If the repeated setup and packaging steps are the bottleneck, then the full pack may save time.
